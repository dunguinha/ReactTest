# ReactTest

## Install

### How to Install?

```bash
nvm use
npm install
```

### Expo-Cli

If it the first time that you are running the application, you need to install expo-cli globally. You can see more [Here](https://docs.expo.dev).

```bash
nvm use
npm install --global expo-cli
```

### How to run?

```bash
nvm use
npm run web
```
